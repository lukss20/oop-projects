Database Sudoku Solver
A Sudoku solver application integrated with a database system, developed to manage, store, and solve Sudoku puzzles efficiently.
Features
- Solve Sudoku puzzles using algorithmic approach
- Store puzzles and solutions in a database
- Retrieve and manage saved puzzles
- Validate Sudoku boards

Technologies
- Language: Java
- Database: MySQL
- Concepts: Data structures, backtracking, database integration

How It Works
The solver uses a backtracking algorithm to fill empty cells while ensuring Sudoku constraints:
- Each number appears once per row, column, and 3x3 grid

Purpose
This project demonstrates integration between **algorithmic problem solving** and **database systems**.

Notes
