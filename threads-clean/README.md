Multithreaded Web Loader
A Java application that using threads and displays progress in a graphical interface.

Features
- Concurrent downloading using multiple threads
- Progress tracking for each task
- Start/stop controls for threads
- Real-time updates in UI

Technologies
- Language: Java
- UI: Java Swing
- Concepts: Multithreading, concurrency

Key Concepts
- Thread lifecycle management
- Synchronization and safe shared state
- Performance optimization using concurrency

Purpose
This project demonstrates practical use of **multithreading** to improve performance in I/O-bound tasks.

Notes
Focus was on thread coordination and avoiding race conditions.
