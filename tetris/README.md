Tetris Game (Java)
A classic Tetris game implementation developed using Java, focusing on game logic, UI interaction, and object-oriented design.
Features
- Real-time block falling and movement
- Row clearing and score tracking
- Collision detection
- Game over conditions

Technologies
- Language: Java
- UI: Java Swing

Key Concepts
- Object-Oriented Programming (OOP)
- Game loop and event handling
- Grid-based logic

Purpose
The goal of this project was to implement a fully functional game while practicing **OOP principles** and **UI programming**.

Notes
This project was developed as part of academic coursework.
